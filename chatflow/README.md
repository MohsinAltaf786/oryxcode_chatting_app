# ChatFlow - Premium Flutter Chat UI Kit

A beautiful, feature-rich Flutter chat application UI kit with modern messaging features.

## ✨ Features

### Core Messaging
- **Modern Chat Interface** - Beautiful chat bubbles with sender/receiver styling
- **Message Reactions** - Quick reactions with 6 default emojis (❤️ 👍 😂 �� 😢 🙏)
- **Reply to Messages** - Swipe-to-reply gesture or long-press menu
- **Message Actions** - Copy, Forward, Star, Delete with long-press
- **Emoji Picker** - Full emoji keyboard integrated into chat input
- **Voice Messages** - Record and send voice messages

### Media Management
- **Media Gallery** - View all shared photos, videos, and documents
- **Filter by Type** - Separate tabs for Photos, Videos, Documents, Links, Audio
- **Grid/List Views** - Toggle between different viewing modes
- **Multi-Selection** - Select and manage multiple media items
- **Full-Screen Viewers** - Image viewer with zoom, Video player, PDF viewer

### Organization
- **Starred Messages** - Bookmark important messages
- **Archived Chats** - Archive conversations
- **Search & Filter** - Find messages and media quickly
- **Storage Management** - Track storage usage

## 🚀 Getting Started

1. Install dependencies: `flutter pub get`
2. Run the app: `flutter run`
