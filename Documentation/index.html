What's New in Version 2.0

The Biggest Update Yet! Version 2.0 brings major enhancements, new screens, and completely redesigned UI/UX.

New Features & Screens

  • Communities Screen: Brand new communities feature to build group-based social interactions within your chat app.
  • Media Viewer Screens: Dedicated screens for viewing different types of media:
    • Audio Player
    • Video Player
    • Image Viewer
    • Document Viewer
  • Emoji Picker: Integrated emoji picker in the conversation screen for expressive messaging.
  • Archived Chats: New archived chats feature to keep your chat list organized.

Redesigned UI/UX

Completely revamped designs for a more modern and intuitive experience:

  • Audio Call Screen
  • Video Call Screen
  • Conversation Screen
  • Status Screen
  • Settings Screen

Technical Improvements

  • Upgraded to Flutter 3.38
  • Performance optimizations
  • Bug fixes and stability improvements

Installation

Make sure you have Flutter installed on your system. If not, follow the instructions here to install Flutter on your preferred platform.

Running the App

  1. Open your favorite IDE (Integrated Development Environment) such as Android Studio, Visual Studio Code, or IntelliJ IDEA.
  2. Navigate to the chat_flow directory using your IDE.
  3. Run flutter pub get in the terminal to install the required dependencies.
  4. Connect your device or start an emulator.
  5. Run flutter run in the terminal to launch the app on your device/emulator.

Following these steps will allow you to set up and run the ChatFlow - Flutter Chat App on your local environment. If you encounter any issues during the installation or running process, please refer to the Flutter documentation or seek assistance from the Flutter community.

Feel free to explore the app and customize it according to your requirements. If you have any questions or feedback, don't hesitate to reach out.
